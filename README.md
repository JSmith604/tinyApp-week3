## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session
- chai
- mocha
- nodemon



## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node expressServer.js` command, or use `npm start` if nodemon is installed.
